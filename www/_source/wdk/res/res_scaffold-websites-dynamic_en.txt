TID_SCAFFOLD_WEBSITES_DYNAMIC_NAME
Create a new dynamic website
TID_SCAFFOLD_WEBSITES_DYNAMIC_DESCRIPTION
This creates the source code of a dynamic website based on the Website Development Kit. The website system supports users, audit-safe logging, sessions, scheduled jobs, the test center, and generic data items.
TID_SCAFFOLD_WEBSITES_DYNAMIC_CONTROL_ROOTURL
Root URL without protocol and sub directory (e.g. "www.example.com")
TID_SCAFFOLD_WEBSITES_DYNAMIC_CONTROL_SITEDIR
Site's sub directory (e.g. "mysite/" if the full URL is "www.example.com/mysite/", leave blank if your site is at the root URL)

TID_SCAFFOLD_WEBSITES_DYNAMIC_CONTROL_WEBSITENAMESHORT

TID_SCAFFOLD_WEBSITES_DYNAMIC_CONTROL_TMPDIR

TID_SCAFFOLD_WEBSITES_DYNAMIC_CONTROL_ADMINEMAIL

TID_SCAFFOLD_WEBSITES_DYNAMIC_CONTROL_AUTHOR
Author's name
TID_SCAFFOLD_WEBSITES_DYNAMIC_CONTROL_PUBLISHER
Publisher
TID_SCAFFOLD_WEBSITES_DYNAMIC_CONTROL_SITETEAM
The site operator team name (e.g. used as the sender name of emails sent by the website)
