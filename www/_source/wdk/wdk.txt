=Website Development Kit (WDK)=
Version 0.0.0.77


by Sascha Wildgrube (sascha@wildgrube.com)
=What is WDK?=
The Website Development Kit is a PHP framework for building
complex web applications.
It addresses numerous issues web application developers have
to take care of. It contains solutions to create multi-lingual
dynamic websites and database wrapping webservices.
